RESEARCH CONTEXT
The old work was dynamic path planning with improved ACO and combined route optimization.
The revised supervisor-directed focus is:

When a route is suddenly blocked, a link is broken, or a protest closes a road in an unplanned megacity like Dhaka, how quickly can vehicles be rerouted through category-wise feasible alternative paths?

This is not only a “best path under normal traffic” problem.
It must model:
- road hierarchy
- route width
- road condition
- vehicle-category feasibility
- pair-level routing
- combined network behavior
- sudden disruption handling
- comparison of three online rerouting cases

VEHICLE CATEGORIES
Category 1: Truck
Category 2: Bus (predefined corridors / allowed bus roads only)
Category 3: Private car, Microbus
Category 4: Rickshaw, Van, Bike

ROUTE HIERARCHY
Main route
Link route
Subroute
Mini route

ACCESS LOGIC
- Not all routes are suitable for all vehicles.
- Truck and bus cannot use every route type.
- Bus must respect predefined bus corridors.
- Private car and microbus can use more routes than truck and bus, but still not every route.
- Rickshaw, van, and bike can use all route types unless an explicit restriction is configured.
- Route width, road condition, and category-wise access must be enforced before a path is accepted.
